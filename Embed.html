How to Embed a Service-Hosted Video into a Website

Step 1: Get the Embed Code

Follow the instructions on the video hosting service to get the embed code for the video you want to embed.

Step 2: Add the Embed Code to Your Website

Copy the embed code provided by the video hosting service and paste it into your website's HTML where you want the video to appear.

Step 3: Adjust the Video Size (Optional)

If needed, adjust the width and height values in the embed code to resize the video on your webpage.
